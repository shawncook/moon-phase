# Moon Phase App

https://moon-phase.herokuapp.com

## Install

```sh
npm install
```

## Usage

First run the Express server:

```sh
npm run start
```

Build javascript and stylesheets:

```sh
npm run build
```

Watch javascript and stylesheets for changes:

```sh
npm run dev
```
